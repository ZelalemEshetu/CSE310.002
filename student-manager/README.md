## 📌 Project Description
The Student Manager is a simple web application built using HTML, JavaScript, and Firebase Firestore.  
It allows users to add, display, and delete student records in real time using a cloud database.

This project demonstrates basic CRUD (Create, Read, Delete) operations using Firebase.

---

## 🚀 Features

- ➕ Add new student (Name, ID, Course)
- 📋 Display all students from Firebase Firestore
- ❌ Delete student records
- 🔄 Real-time database interaction
- 🌐 Cloud-based storage using Firebase

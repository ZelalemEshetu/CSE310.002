# Todo App (Flutter)

## Project Overview

This is a simple Todo application built using Flutter. The app allows users to add tasks, mark tasks as completed, and delete tasks. It demonstrates basic state management using `setState` and clean UI design in Flutter.

## Repository Structure

This project is inside the `CSE310.002` repository:

```text id="repo1"
CSE310.002/
└── todo-app/
    ├── lib/
    │   ├── main.dart
    │   ├── models/
    │   │   └── task.dart
    │   └── screens/
    │       └── todo_page.dart
```

## Features

* Add new tasks
* Mark tasks as completed using checkbox
* Delete tasks from the list
* Dynamic UI updates using `setState`
* Simple and user-friendly interface

## How to Run the Project

1. Install Flutter SDK on your system
2. Clone the repository:

   ```
   git clone https://github.com/ZelalemEshetu/CSE310.002.git
   ```
3. Navigate into the project folder:

   ```
   cd CSE310.002/todo-app
   ```
4. Get dependencies:

   ```
   flutter pub get
   ```
5. Run the app:

   ```
   flutter run
   ```
